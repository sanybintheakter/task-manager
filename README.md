# A custom task management app built with Next.js, React and Tailwind CSS.

## Features

- Add new tasks
- Toggle tasks between active and completed
- Delete individual tasks
- Filter by All, Active and Done
- View live task statistics
- Clear all completed tasks
- Persist tasks with localStorage

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS v4

## How to Run

bash
npm install
npm run dev
